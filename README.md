# Web Content API

![](https://img.shields.io/badge/maven-3.6.3-blue.svg)
[![Maven Central](https://maven-badges.herokuapp.com/maven-central/us.careydevelopment.util/web-content-api/badge.svg)](https://search.maven.org/artifact/us.careydevelopment.util/web-content-api/1.1.0-RELEASE/jar)



This package handles CRUD activities for web content objects.


## Latest Release
You can pull it from the central Maven repositories:

```xml

    <groupId>us.careydevelopment.util</groupId>
    <artifactId>web-content-api</artifactId>
    <version>1.0.0-RELEASE</version>
 </dependency>
```
